                    ✅ REGISTRATION REDIRECT FIXED!
================================================================================

THE PROBLEM:
After registration, users were redirected to login instead of dashboard.

THE CAUSE:
The updateCurrentUser function was setting the user but NOT setting
isAuthenticated to true, so ProtectedRoute thought user wasn't logged in.

THE FIX:
Updated src/contexts/UserContext.tsx:

    const updateCurrentUser = (user: User) => {
      setCurrentUser(user);
      setIsAuthenticated(true);  // ← ADDED THIS LINE
    };
